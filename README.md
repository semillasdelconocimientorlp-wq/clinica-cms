# Clinica CMS

This repository contains the Clinica CMS project, which is designed for managing clinical information in a comprehensive manner.

## Features
- User management
- Patient records
- Appointment scheduling
- Treatment history

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/semillasdelconocimientorlp-wq/clinica-cms.git
   ```
2. Navigate to the project directory:
   ```bash
   cd clinica-cms
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
